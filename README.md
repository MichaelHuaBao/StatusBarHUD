# XMGStatusBarHUD
一个非常简单易用的状态栏指示器小框架

## 显示成功信息
```objc
[XMGStatusBarHUD showSuccess:@"加载成功!"];
```

## 显示失败信息
```objc
[XMGStatusBarHUD showError:@"加载失败!"];
```

## 显示普通信息
```objc
[XMGStatusBarHUD showMessage:@"没有什么事!!!!"];
```

## 显示加载信息
```objc
[XMGStatusBarHUD showLoading:@"正在加载中..."];
```

## 隐藏
```objc
[XMGStatusBarHUD hide];
```
